# 个人作品集网站

**Full Stack 学习计划 - Day 1 项目**

## 📋 项目信息

- **类型**: 响应式单页应用（SPA）
- **技术栈**: React + Vite
- **目标**: 掌握前端基础和现代开发流程

## 🚀 快速开始

### 安装依赖
```bash
cd /root/.openclaw/workspace-dev/projects/portfolio
npm install
```

### 开发模式
```bash
npm run dev
```
访问: http://localhost:3000

### 构建生产版本
```bash
npm run build
```

### 预览生产构建
```bash
npm run preview
```

## 📁 项目结构

```
portfolio/
├── src/
│   ├── App.jsx          # 主应用组件
│   ├── main.jsx          # 应用入口
│   ├── index.css         # 全局样式
│   └── assets/           # 静态资源
├── index.html            # HTML 模板
├── vite.config.js        # Vite 配置
└── package.json          # 项目配置
```

## ✨ 功能特性

- [x] 响应式导航（移动端适配）
